### CROSS-LAB CONVERGENCE (highest pile-on signal)
1. **Attention fragmented — 3 camps, NO consensus:** linear/hybrid (Gated DeltaNet 3:1, Kimi Linear) vs KV-compression (MLA + DeepSeek Sparse Attention) vs deliberate conservatism (MiniMax M2/M3 **reverted to full/exact-softmax** because linear degraded reasoning/multi-turn). The DISAGREEMENT is the story.
2. **Muon optimizer displacing AdamW at trillion scale** (Kimi K2 1T, DeepSeek V4, GLM-5 744B; MuonClip 15.5T tokens zero spikes; follow-up MONA 2605.26842). Quietly huge, under-covered.
3. **Agentic RL on massive interactive environments = table stakes** (Qwen3.5, MiniMax, Grok 4, OLMo RLVR/RL-Zero, GPT-5.5). Academic wave: AgentRL (2510.04206), ProRL-Agent (2603.18815).
4. **Test-time compute productizing into parallel-reasoning / multi-agent debate** (Grok 4.20, Muse Spark, Gemini Deep Think, Anthropic effort dial).

Freshness VERY HIGH.
- **B2 Agent long-term memory (post-RAG) — SURGING.** Agentic Memory (2601.01885), Memory in the LLM Era (2604.01707), MemCoT (2604.08216), Security of Long-Term Memory in LLM Agents (2604.16548 — mature enough for attack-surface lit); memory-as-weights (Meta Memory Layers 2412.09764, ICLR 2026).

Discourse shifted "are they real?" → "controllability + evaluation." VERY HIGH.

## NeurIPS 2026 prediction candidates
- **C1 Test-Time Training / adaptation as first-class method.** Novel (updates weights DURING inference vs frozen+RAG). High-prob (labs ship productized cousin; ARC top tier relies on it; theory bridge 2602.21204). **Conf 80%.** Criterion: ≥15 accepts (main+wkshp) w/ TTT/test-time adaptation/continual-at-inference central + TTT-adjacent workshop runs.
- **C2 Agentic RLVR in large interactive-environment suites.** Novel (multi-turn env-grounded RL w/ verifiable not learned reward). High-prob (every frontier model validated; open wave underway). **Conf 82%.** Criterion: RLVR/agentic-RL dominant post-training theme by count + ≥1 agentic-RL benchmark standard across ≥10 accepts.
- **C3 World models for planning/agents/robotics (latent-JEPA + video-as-WM).** Novel (generative model as SIMULATOR for planning/RL; physics-faithfulness central). High-prob (labs Genie3→Waymo + replicable JEPA + critical eval lit + standing workshop). **Conf 75%.** Criterion: WM workshop at NeurIPS 2026 + ≥20 main-track WM-for-planning/robotics papers + "physics faithfulness/controllability" named sub-theme.
- **C4 AI-for-math w/ formal verification (LLM+Lean, open-problem-solving).** Novel (machine-verified GENUINELY NOVEL theorems vs informal CoT). High-prob (AlphaProof Nexus Erdős; FrontierMath v2; open-prover race; IMO-2026-during-review tailwind). **Conf 70%.** Criterion: MATH-AI/formal-reasoning workshop + ≥1 headline newly-formally-verified open-problem result + FrontierMath/Lean cited across math track.
- **C5 Elastic/adaptive test-time compute as controllable trained dial (parallel reasoning + multi-agent debate).** Novel (trained dialable parallel/breadth + token-efficiency vs fixed long chains). High-prob (4 labs independently shipped H1 2026). **Conf 68%** (overlaps C1/C2). Criterion: ≥10 accepts centrally on adaptive/parallel test-time-compute allocation or multi-agent-debate efficiency.
- **Dark horse: Muon-family optimizers.** First serious AdamW challenger at trillion-param production. **Conf 55%** (optimizer papers skew ICLR/ICML > NeurIPS). Criterion: ≥8 accepts on Muon/orthogonalized-update optimizers.

**Top 3 to bet on:** TTT/adaptation (freshest inflection); agentic RLVR (most lab-validated); world models (cleanest surging-in-both-worlds). Most OVERRATED: energy-based/non-backprop. Most UNDER-COVERED: Muon sweep + attention-fragmentation (MiniMax linear-attention retreat = contrarian counter-signal to SSM hype).
